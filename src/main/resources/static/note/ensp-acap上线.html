ensp wifi配置笔记 ​

1.让ap上线 ​

（1）ap通过ac的dhcp发放获取ip地址

​ 创建vlan 在ac的vlanif上面配置dhcp ：dhcp select interface

（2）通过capwap隧道上线

​ capwap source intface vlanif 100 //通过100管理这条隧道

​ ap-group name hcia //创建ap组来统一管理

（3）让ap设备进入ap组管理

​ ap-id 1 ap-mac MAC地址 //apmac认证

​ ap-name ap1 //ap设备命名ap1

​ ap-group hcia //把设备分到组里面统一管理

2.配置无线参数 ​

ssid模板：

ssid-profile name wlan-ssid

ssid huawei

安全模板:

security-profile name wlan-sec

security wpa-wpa2 psk pass-phrass 密码 aes

vap模板:

vap-profile name wlan-vap

service-vlan vlan-id 10

加组

ssid-profile wlan-ssid

security-profile wlan-sec

通过ap-grout组统一管理调用vap模板:

ap -group name hcia

vap-profile wlan-vap wlan 1 radio all
